# 🖼 Image To Prompt Generator

A modern Image To Prompt Generator built with HTML, CSS, and JavaScript. Upload images, preview them instantly, generate AI-style prompts, and copy prompts with a clean and responsive interface.

---

## 🚀 Features

### 📤 Image Upload

- Upload images from your device
- Instant image preview
- Drag & click upload experience

### 🖼 Live Preview

- Real-time image display
- Responsive preview container
- Modern gallery-style layout

### ✨ Prompt Generator

- Generate AI-style image prompts
- Professional prompt templates
- One-click prompt creation

### 📋 Copy Prompt

- Copy generated prompts instantly
- Clipboard support

### 🗑 Clear Workspace

- Remove image preview
- Reset prompt output
- Start fresh with one click

### 🎨 Modern UI

- Dark Theme
- Responsive Design
- Glassmorphism Inspired Layout
- Mobile Friendly
